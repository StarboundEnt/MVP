## Store Listing Optimization Tips

### Title Optimization
- Include primary keyword: "Starbound - Habit & Wellbeing Tracker"
- Keep under 30 characters for visibility

### Description Best Practices
- Lead with benefits, not features
- Use emojis sparingly for readability
- Include keywords naturally
- Add social proof (testimonials, ratings)
- Clear call-to-action

### Visual Assets
- Use consistent branding and colors
- Show actual app interface (no mockups)
- Include diverse users (if showing people)
- Test on different screen sizes
- A/B test different screenshot orders
